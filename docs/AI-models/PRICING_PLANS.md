## ❓ Frequently Asked Questions

### What happens after my free trial ends?
Your account remains active but you must upgrade to a paid plan to continue processing audio. Your data is preserved for 30 days.

### Can I change plans anytime?
Yes! Upgrade or downgrade anytime. Changes take effect immediately. Downgrades are prorated.

### What counts as "1 AI use"?
Running AI duplicate detection, transcription, or PDF comparison on one audio file = 1 AI use (approximately £0.60-£1.20 depending on length).

### Do unused AI credits roll over?
No, AI budgets reset monthly. However, Enterprise plans can request rollover options.

### Can I buy extra AI credits?

## 🔗 Linking to Stripe Checkout

When a user clicks a plan button, redirect to Stripe Checkout:

```javascript
// frontend/src/utils/stripe.js
import { loadStripe } from '@stripe/stripe-js';

const stripePromise = loadStripe('pk_live_YOUR_STRIPE_PUBLIC_KEY');

export const openCheckout = async (planId) => {
  const stripe = await stripePromise;
  const token = localStorage.getItem('authToken');
  
  // Call your backend to create checkout session
  const response = await fetch('/api/auth/create-checkout-session/', {
    method: 'POST',
    headers: {
      'Authorization': `Token ${token}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ plan: planId })
  });
  
  const session = await response.json();
  
  // Redirect to Stripe Checkout
  const { error } = await stripe.redirectToCheckout({
    sessionId: session.id
  });
  
  if (error) {
    console.error('Stripe error:', error);
  }
};
```

**Then in PricingPlans.jsx:**

```javascript
import { openCheckout } from '../utils/stripe';

const handlePlanSelect = (planId) => {
  if (planId === 'free-trial') {
    navigate('/signup');
  } else if (planId === 'enterprise') {
    navigate('/contact-sales');
  } else {
    openCheckout(planId);
  }
};
```
